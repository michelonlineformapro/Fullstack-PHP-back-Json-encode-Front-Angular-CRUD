Explications :

Ce site utilise une API REST, le backend est un CRUD classique PHP, chaque opération est encodé en json avce la fonction native PHP : json_encode, les opérations sont les suivante :

  • EXPLICATION :
  • 1 - Angular recupère les fichiers PHP CRUD encoder en Json
  • 2 - Angulare utilise des sercices pour comminiquer avec le backend via les opération REST
  • 3 - Les opérations REST sont les suivante GET, POST, PUT-PATCH & DELETE
  • 4 - Ce sont des requète HTTP qui retourne un status (200 et 400)
  • 5 - Les composants Angular appels les services et leurs methodes (fonctions)
  • 6 - Les fichiers HTML des composant appels également des fonctions et sont capable de réaliser des condition *ngIf, des boucles *ngFor, des formulaires, etc...
